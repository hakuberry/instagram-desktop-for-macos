## Installation

Generate DMG with Electron Forge

```bash
  cd <instagram-desktop-for-macos repository>
  npm run make
```
Open Instagram-1.0.0-arm64.dmg and install Instagram app

```bash
  <instagram-desktop-for-macos repository>/out/make
```
